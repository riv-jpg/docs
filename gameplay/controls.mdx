Stubly Baseball supports three distinct control schemes so you can play the way you're most comfortable — on your phone, at your desk, or on the couch with a controller. Every mechanic across batting, pitching, fielding, and menu navigation has a mapped input for each platform. Use the tabs below to find your control scheme.

<Tip>
  You can remap keyboard keys and gamepad buttons anytime in **Settings → Controls**. On mobile, you can reposition touch zones to match your grip style in **Settings → Controls → Touch Layout**. Batting aim sensitivity and pitch zone sensitivity can also be tuned independently per control scheme — dial them in for your playstyle before your first ranked match.
</Tip>

<Tabs>
  <Tab title="Mobile Touch">
    Stubly Baseball on mobile uses tap, hold, swipe, and on-screen joystick gestures. All interactive zones are repositionable in **Settings → Controls → Touch Layout**.

    ### Batting

    | Action | Gesture |
    |--------|---------|
    | Normal Swing | Tap the screen |
    | Power Swing | Hold the swing button for 0.3 seconds, then release |
    | Slap Hit | Swipe left or right quickly on the batting zone |
    | Bunt | Tap the **Bunt icon** on the HUD before the pitch |
    | Aim Pull Side | Swipe the directional joystick left before swinging |
    | Aim Opposite Field | Swipe the directional joystick right before swinging |

    ### Pitching

    | Action | Gesture |
    |--------|---------|
    | Select Pitch Type | Swipe up on the pitch wheel to open; tap your pitch |
    | Aim Location | Drag the crosshair across the zone grid |
    | Throw Pitch | Tap the **Throw button**; power meter fills automatically |
    | Check Runner | Tap the **Runner icon** beside the base indicator |

    ### Fielding

    | Action | Gesture |
    |--------|---------|
    | Move Player | Drag the on-screen joystick |
    | Routine / Running Catch | Tap the **Catch button** |
    | Diving Catch | Double-tap the **Dive button** |
    | Jump Catch (at wall) | Tap **Jump + Catch** buttons simultaneously |
    | Quick Throw | Tap the **base icon** (1B, 2B, 3B, or Home) |
    | Strong Throw | Hold the **Throw button**, release in the green zone |

    ### Menu Navigation

    | Action | Gesture |
    |--------|---------|
    | Navigate Cards | Swipe left or right |
    | Enter Mode / Select | Tap the card |
    | Return to Main Menu | Tap the **back arrow** |
    | Open Settings | Tap the **gear icon** (top-right corner) |
  </Tab>

  <Tab title="PC Keyboard">
    The PC keyboard scheme uses letter keys for actions and arrow keys for navigation. All keys are remappable in **Settings → Controls**.

    ### Batting

    | Action | Key |
    |--------|-----|
    | Normal Swing | `A` |
    | Power Swing | Hold `A` (0.3s), then release |
    | Slap Hit | `S` + `A` together |
    | Bunt | `B` |
    | Aim Pull Side | Hold `Left Arrow` before swinging |
    | Aim Opposite Field | Hold `Right Arrow` before swinging |

    ### Pitching

    | Action | Key |
    |--------|-----|
    | Select 4-Seam Fastball | `1` |
    | Select 2-Seam / Sinker | `2` |
    | Select Curveball | `3` |
    | Select Slider | `4` |
    | Select Changeup | `5` |
    | Select Cutter | `6` |
    | Aim Location | Move the **mouse** over the zone grid |
    | Throw Pitch | `Space` |
    | Check Runner | `R` |

    ### Fielding

    | Action | Key |
    |--------|-----|
    | Move Player | `W` `A` `S` `D` |
    | Routine / Running Catch | `F` |
    | Diving Catch | `G` |
    | Jump Catch (at wall) | `F` + `Space` together |
    | Throw to First Base | `Q` |
    | Throw to Second Base | `E` |
    | Throw to Third Base | `R` |
    | Throw to Home Plate | `T` |
    | Strong Throw | Hold throw key; release in the green zone |

    ### Menu Navigation

    | Action | Key |
    |--------|-----|
    | Navigate Menus | Arrow Keys |
    | Select / Confirm | `Enter` |
    | Back / Cancel | `Escape` |
    | Open Settings | `Escape` from the main menu |
  </Tab>

  <Tab title="Gamepad (Xbox / PS / Switch Pro)">
    Stubly Baseball fully supports Xbox, PlayStation, and Nintendo Switch Pro controllers. Button labels below show **Xbox / PS** equivalents side by side. All bindings are remappable in **Settings → Controls**.

    ### Batting

    | Action | Xbox | PS |
    |--------|------|----|
    | Normal Swing | `A` | `Cross` |
    | Power Swing | Hold `A` (0.3s), release | Hold `Cross` (0.3s), release |
    | Slap Hit | `B` + `A` together | `Circle` + `Cross` together |
    | Bunt | `X` | `Square` |
    | Aim Pull Side | Left Stick ← (hold before swing) | Left Stick ← (hold before swing) |
    | Aim Opposite Field | Left Stick → (hold before swing) | Left Stick → (hold before swing) |

    ### Pitching

    | Action | Xbox | PS |
    |--------|------|----|
    | Open Pitch Wheel | Flick Left Stick | Flick Left Stick |
    | Aim Location | Right Stick | Right Stick |
    | Throw Pitch | `RT` | `R2` |
    | Check Runner | `LB` | `L1` |
    | Waste Pitch (off zone) | Hold `LT` + aim | Hold `L2` + aim |

    ### Fielding

    | Action | Xbox | PS |
    |--------|------|----|
    | Move Player | Left Stick | Left Stick |
    | Routine / Running Catch | `A` | `Cross` |
    | Diving Catch | `B` | `Circle` |
    | Jump Catch (at wall) | `A` + `Y` together | `Cross` + `Triangle` together |
    | Throw to First Base | `A` | `Cross` |
    | Throw to Second Base | `X` | `Square` |
    | Throw to Third Base | `Y` | `Triangle` |
    | Throw to Home Plate | `B` | `Circle` |
    | Strong Throw | Hold face button; release in green zone | Hold face button; release in green zone |
    | Right Stick Throw (advanced) | Click `RS` after catch | Click `R3` after catch |

    ### Menu Navigation

    | Action | Xbox | PS |
    |--------|------|----|
    | Navigate | Left Stick or D-Pad | Left Stick or D-Pad |
    | Select / Confirm | `A` | `Cross` |
    | Back / Cancel | `B` | `Circle` |
    | Open Pause Menu | `Start` / `Menu` | `Options` |
  </Tab>

## Customizing Your Controls

Every control scheme is fully adjustable so the game fits exactly how you want to play.

<CardGroup cols={3}>
  <Card title="Remap Keys & Buttons" icon="keyboard">
    Navigate to **Settings → Controls** and select your active scheme. Tap or click any action to reassign it to a new key, button, or gesture. Changes apply immediately and persist across sessions.
  </Card>
  <Card title="Reposition Touch Zones" icon="hand-pointer">
    Mobile players can drag any HUD element — swing zone, joystick, pitch wheel, base icons — to a new screen position in **Settings → Controls → Touch Layout**. Resize zones to match your thumb reach.
  </Card>
  <Card title="Adjust Aim Sensitivity" icon="sliders">
    Tune the batting aim speed and pitch zone drag speed separately per control scheme in **Settings → Controls → Sensitivity**. Lowering sensitivity gives you finer control for precision pitch placement.
  </Card>
</CardGroup>
